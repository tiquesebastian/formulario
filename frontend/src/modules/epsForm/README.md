# Comentarios de la carpeta epsForm

Contiene toda la lógica de interfaz del formulario EPS en piezas reutilizables.

- `components/`: piezas UI pequeñas.
- `config/`: catálogos y configuraciones estáticas.
- `schema/`: validación y tipos del formulario.
- `sections/`: secciones visuales del formulario.
- `types/`: tipos auxiliares.
- `utils/`: utilidades de apoyo.
